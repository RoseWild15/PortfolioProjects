## Amazon Web Scraping Project

Here you'll find my code for the an Amazon Web Scraper project, in its current state it is setup to track the price of a funny data analyst tshirt and send myself an email whenever it dips below price however this could easily be expanded/altered to scrape for a set of products and track their prices etc.

## Tools/Skills Used
• Python

• Pandas

• BeautifulSoup & Requests

## Project Goal
Scrape Amazon for a product price and create an email alert to alert when the product dropped below its price.
