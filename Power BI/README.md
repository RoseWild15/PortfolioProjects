## Data Professional Survey Dashboard

Here you'll find the dashboard I created in Power BI to identify various trends and patterns from a data professional survery. There is also a screenshot of the dashboard for quick viewing. 

## Tools Used
• Power BI

• Power Query

• DAX

## Project Goal
Analyze and create a dashboard reflecting trends & patterns from data professional survey.

## Key Insights (Global)
• The job title with the highest average salary is Data Scientists.

• Most data professionals favor Python as their favorite programming language.

• Overall happiness with work life balance scored higher than salary satisfaction.
